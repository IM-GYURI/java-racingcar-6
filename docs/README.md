### 입력 관련
- 자동차 이름을 입력받아 자동차 생성
  - 쉼표로 구분
  - 이름이 5자 이하인지 확인
  - 중복 제거
  - 빈 값이 있는지 확인
- 이동을 시도할 횟수 입력
  - 숫자인지 확인
  - 양수인지 확인

### 게임 진행
- 0에서 9 사이의 랜덤값 생성
- 랜덤값이 4 이상이면 자동차를 전진
- 매 시도마다 진행 상태 출력
